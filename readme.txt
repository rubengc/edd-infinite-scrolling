Product lists infinite scrolling for Easy Digital Downloads.

== Description ==

EDD Infinite Scrolling is a very easy-to-use plugin for Easy Digital Downloads. You just need to install it and it's ready to work.

EDD Infinite Scrolling loads the content of your product lists in a sequential way: think about Twitter or Facebook, two navigation systems that now influence how we browse the web. EDD Infinite Scrolling adds this idea for an immediate and direct consultation of the products of your Easy Digital Downloads site!

No more loading times caused by the change of the page! Everything will come in a single solution, where users, even from a mobile device, will be able to simply scroll down and get the details they seek right away, without being afraid to lose them loading a new page.

= Features of EDD Infinite Scrolling =

* Option to enable it by default in all [downloads] and [edd_downloads] of your site.
* Entrance transition for new loaded products.

If you want to boost your EDD Infinite Scrolling, you can also visit our page and explore the [official page](http://tsunoa.com/downloads/edd-infinite-scrolling/) of the premium version, which includes all the following features to improve at the best your Easy Digital Downloads store search.

= Premium features of EDD Infinite Scrolling =

* Different content loading functions:
    * Infinite scrolling.
    * Load more button.
    * Ajax pagination.
* Ability to set different content loading to each product list.
* Options to set up returned content on last loaded page with the possibility of set up different ones to each loading function.
* Different entrance/exit transition to each loading function.
* Full control to customize the load more button:
    * Button label and visual controls to match with your theme style (colors, borders, styles).
* Options to force or disable plugin css resources.

For a more detailed list of options and features of EDD Infinite Scrolling, please read the [official documentation](http://tsunoa.com/docs/edd-infinite-scrolling/ "EDD Infinite Scrolling official documentation").

= Compatibility with other plugins =

EDD Infinite Scrolling has been tested with the following plugins:

* [**EDD Ajax Search**](http://wordpress.org/plugins/edd-ajax-search/)
* [**EDD Quick view**](https://wordpress.org/plugins/edd-quick-view/)

Nevertheless, it could be compatible with many other Easy Digital Download plugins that have not been tested yet. If you want to inform us about compatibility with other plugins, please, [email us](mailto:contact@tsunoa.com "Tsunoa").

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
